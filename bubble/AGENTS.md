Note to LLM agents reading this: The files in this folder are:
- `playgram.bubble.json` — the entire exported Bubble.io project as JSON
- `split_bubble.py` — a script that splits the `playgram.bubble.json` file into smaller files
- `playgram_split` — a folder that contains the split files. These will be useful for understanding the structure of the Bubble.io project whenever you need it.

The first of the above, as well as some of the files in the `playgram_split` folder, tend to be large. Don’t read them before pre-checking their line count. Avoid reading the entire project JSON at all, as all the info you might need is already in the split files.
